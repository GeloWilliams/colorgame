What is RGB?

Traditionally, we take a mixture of red, blue, and yellow to represent all hues within the color range; these are what we call the "primary colors".

It is from this basis that we can produce any color we fancy. The RGB color model, on the other hand, takes this idea and flips it on its head.

r = Red | g = Green | b = Blue
The syntax for RGB colors is:
rgb(#, #, #)

So YELLOW has been replaced by GREEN!

Although seemingly based on our perception of color, the RGB model works best for displaying electronic images on screen.

Known as "channels", each letter in the RGB model represents the degree of its corresponding hue: r, g, and b.

Let's start by understanding the range of values.
If we want to create black, all channels would need a value of 0:

Black = rgb(0, 0, 0)

For white, all channels need a value of 255:

White = rgb(255, 255, 255)

This means that any color we want to create falls between 0 and 255. For a pure gray, we can use:

Gray = rgb(128, 128, 128)

Keep in mind that the higher the number, the brighter the color.
